## The All Jobs list

Find any job in your company from one screen using search, date filters, and status badges. The list helps you scan key details quickly, spot issues, and open the right job without digging through separate views.

## Prerequisites

<Callout kind="info">

The **All Jobs** list is available to **Owners** and **Managers**. Techs see a filtered version called **My Assigned Jobs** that shows only their assigned work. See [Using My Assigned Jobs](/jobs/my-assigned-jobs) for details.

</Callout>

## What you see

Each row shows the job's most important details at a glance.

| Column | What It Shows |
| --- | --- |
| Job Details | The job title and its ID number, such as `#1234`. |
| Client | The client name for the job. |
| Assigned To | The primary team member's name and role. Shows **Unassigned** if no one is assigned, or `+N` if multiple people are assigned. |
| Date and Time | The scheduled date and time. If a duration is set, shows the time range, such as `9:00 AM - 10:30 AM`. |
| Status | The job status badge — **Scheduled**, **Ready**, **In Progress**, **Completed**, or **Archived** — plus an **Issue** badge if the job has an open issue flag. |

## Using filters

Use the filter bar at the top of the list to narrow down jobs.

<Steps>
  <Step title="Search by job title, ID, or client name" title-type="p">

  Type in the search box to find jobs by title, ID number, or client name.

  </Step>
  <Step title="Filter by status" title-type="p">

  Use the status dropdown to show only jobs in a specific status. The dropdown includes all five statuses plus a **Has Issue** option that shows only jobs with an open issue flag.

  </Step>
  <Step title="Filter by date range" title-type="p">

  Use the date range fields to show only jobs scheduled within a specific time period. Set both a start and end date to limit the range, or set only a start date to see everything from that date forward.

  </Step>
  <Step title="Browse the results" title-type="p">

  The list shows up to 50 jobs, sorted by newest first. Click any job row to open its detail page.

  </Step>
</Steps>

<Callout kind="info">

The **OVERDUE** chip appears next to the date on jobs with a past scheduled date that are not **Completed** or **Archived**. Jobs scheduled for today are not marked as overdue.

</Callout>

</Callout>
