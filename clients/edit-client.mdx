## Edit a client

Keep client profiles current as contact info and service locations evolve. Open the client record, update the details that changed, and save to return to the client detail page.

<Callout kind="info">

Sign in as an **Owner** or **Manager**. The edit client page requires permission to view clients and edit clients.

</Callout>

<Steps>
  <Step title="Open the client detail page" title-type="h3">

  Go to the **Clients** page and click the client you want to update. The **Edit Client** button appears only on the client detail page, not in the client list.

  </Step>
  <Step title="Click Edit Client" title-type="h3">

  Select **Edit Client** to open the edit form.

  </Step>
  <Step title="Update the client information" title-type="h3">

  Change any field that needs to be updated. The form uses the same fields as **Add Client** and opens with the current values already filled in.

  Update these fields as needed:

  - business name
  - primary contact
  - email
  - phone
  - street address
  - city
  - state
  - zip code
  - notes

  </Step>
  <Step title="Save your changes" title-type="h3">

  Click **Save Changes**. A success message appears, and NoSavvy returns you to the client detail page.

  </Step>
</Steps>

<Callout kind="info">

Changes to client information appear on all future jobs for that client. Existing jobs keep their original job details, but the client section on those jobs reflects the updated client information.

</Callout>

<Callout kind="info">

The edit page does not include a delete option. To delete a client, use the **Delete** button on the client detail page.

</Callout>

<Callout kind="tip">

**Where to go next**

- Review past activity on the [Client History](/clients/client-history) page.
- Create another record from the [Add Client](/clients/add-client) page.

</Callout>
